Classes

Classes are "types" of objects that share common variables, methods, or other properties. To take a real world example, we could create a "dog" object that belongs to the "pet" class.

class NameOfClass
  def initialize(x)
    @x = x
  end
end

class NameOfClass < SuperClass
  # this is the syntax for class inheritance.
end

Blocks, procs, lambdas

A block is a chunk of code that does something. It can be separated by curly brackets {} or "do" and "end". Some methods take blocks as arguments; for example each, times, and select. You can write methods that accept blocks using "yield":

def method(n)
  yield n
end

Passes "n" to the block. yield can also be standalone.

A proc is kind of like a "saved block." It can be assigned to a variable then passed to methods that take blocks. "&" is used as a prefix to convert a proc into a block:

cube = Proc.new { |x| x ** 3 }
[1, 2, 3].collect!(&cube)

Lambdas are more or less identical to procs, with a key difference: when a lambda returns, it passes control back to the calling method; when a proc returns, it "breaks" without finishing the rest of the method.
